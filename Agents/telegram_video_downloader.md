# 🤖 Агент: Telegram Video Downloader Lead

## 🎯 Цель
Разработка и деплой телеграм-бота для скачивания видео из социальных сетей (Instagram, Twitter, TikTok) на VPS.

## 📋 Инструкции
1. Использовать `aiogram` для интерфейса бота.
2. Использовать `yt-dlp` для скачивания контента.
3. Обеспечить автоматическую очистку скачанных файлов после отправки.
4. Задеплоить через Docker для стабильности и изоляции.

## 🛠 Технологии
- Python 3.11
- aiogram 3.x
- yt-dlp
- ffmpeg
- Docker & Docker Compose
